# Rayna Tours - ChatGPT App SDK

A ChatGPT App SDK project that renders beautiful tour cards directly inside ChatGPT conversations. Built with Python (FastMCP) + React/TypeScript UI widgets.

## Prerequisites

- **Python** 3.11+
- **Node.js** 18+ & npm
- **uv** (Python package manager) — [install](https://docs.astral.sh/uv/getting-started/installation/)
- **ngrok** (for exposing local server to ChatGPT) — [install](https://ngrok.com/download)

## Setup

### 1. Clone & enter the project

```bash
git clone <repo-url>
cd raynachatgptsdk-app
```

### 2. Environment variables

```bash
cp .env.example .env
```

Edit `.env` and set your OpenAI verification token:

```env
RAYNA_API_BASE_URL=https://data-projects-flax.vercel.app/api
PORT=8000
OPENAI_VERIFICATION_TOKEN=your-token-here
```

### 3. Build React UI widgets

```bash
cd web
npm install
npm run build
cd ..
```

### 4. Install Python dependencies

```bash
uv sync
```

Or with pip:

```bash
pip install -r requirements.txt
```

### 5. Start the server

```bash
uv run python server.py
```

Or without uv:

```bash
python server.py
```

Server starts at `http://localhost:8000`

You should see:

```
============================================================
  Rayna Tours MCP Server with React UI
============================================================

  Endpoints:
  - MCP:    http://0.0.0.0:8000/mcp
  - Health: http://0.0.0.0:8000/health
  - Info:   http://0.0.0.0:8000/info
============================================================
```

## Connect to ChatGPT

1. Start ngrok:

```bash
ngrok http 8000
```

2. Copy the ngrok URL (e.g. `https://abc123.ngrok-free.app`)

3. In ChatGPT → Settings → MCP Servers → Add your server:
   - URL: `https://abc123.ngrok-free.app/mcp`

## Tools

| Tool | Description | Widget |
|------|-------------|--------|
| `show-tours` | Search & display tours/activities as visual cards | Tour card grid |
| `show-tour-detail` | Full tour details with inclusions, exclusions, rating | Tour detail page |
| `compare-tours` | Side-by-side comparison of 2-4 tours | Comparison table |
| `show-holiday-packages` | Holiday & vacation packages | Tour card grid |
| `get-visa-info` | Visa requirements, fees, processing time | Text response |
| `show-yachts` | Yacht & luxury boat experiences | Tour card grid |

## Endpoints

| Path | Description |
|------|-------------|
| `/mcp` | MCP protocol (Streamable HTTP) |
| `/health` | Health check |
| `/info` | Server info |
| `/.well-known/openai-apps-challenge` | OpenAI verification |

## Project Structure

```

├── server.py          # MCP server — tool handlers, widgets, routing
├── rayna_utils.py     # API client, data formatting, enriched-feed helpers
├── web/               # React UI widgets
│   ├── src/           # TypeScript source (TourList, TourDetail, TourCompare)

├── .env.example       # Environment variable template
├── pyproject.toml     # Python project config (uv)
└── requirements.txt   # Python dependencies (pip)
```

## Deploy to Vercel

### 1. Install Vercel CLI

```bash
npm install -g vercel
```

### 2. Build React UI first

```bash
cd web && npm install && npm run build && cd ..
```

### 3. Deploy

```bash
vercel
```

Follow the prompts. On first deploy, select your project settings.

### 4. Set environment variables

```bash
vercel env add OPENAI_VERIFICATION_TOKEN
vercel env add RAYNA_API_BASE_URL
```

Or set them in the Vercel dashboard: **Project → Settings → Environment Variables**

### 5. Redeploy with env vars

```bash
vercel --prod
```

### 6. Connect to ChatGPT

Use your Vercel URL in ChatGPT MCP settings:

```
https://your-project.vercel.app/mcp
```

> **Note:** Vercel serverless functions have a 10s timeout on Hobby plan and 60s on Pro. If tool calls timeout, consider Railway or Render for long-running server support.
